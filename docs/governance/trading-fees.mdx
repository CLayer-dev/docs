## How It Works

**Phased fee reduction** with automated smart contract execution:

1. **0-300 ETH**: 5% fee → Bootstrap development
2. **300-500 ETH**: 4% fee → Growth phase  
3. **500-2000 ETH**: 3% fee → Scaling phase
4. **2000+ ETH**: 0% fee → Fee-free forever
